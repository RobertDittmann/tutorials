# tutorial-tibco-ems
Required TIBCO Enterprise Message Service - Community Edition. Can be downloaded from: https://www.tibco.com/products/tibco-messaging/downloads 

Useful TIBCO EMS commands: https://haritibcoblog.com/2015/05/23/tibco-ems-administration-commands-some-important-ones/ 

Proper jar's from TIBCO EMS distribution should be added into the project. It is:
* jms-2.0.jar
* tibjms.jar

JAR's can be found under TIBCO installation directory, for instance: C:\tibco\ems\8.4\lib
